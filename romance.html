Romantic Poem Generator

Step 1: pick a corpus file
any plaintext file such as .txt will do -- no .txt files? copy a document, blog post, news story, or lyrics page from the web and paste into a text editor such as Notepad (Windows) or vim (Unix) and simply save it!
Step 2: select how many lines of poetry to wish to generate from the drop-down menu
Step 3: press the button to generate your poem!
NOTE: Apparently only Opera resets the drop-down menus upon re-loading -- in other browsers the menus will still show a value but program will not store the value until changed
